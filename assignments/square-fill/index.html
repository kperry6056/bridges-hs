Description

The BRIDGES library contains a class called ColorGrid that allows one to create simple games based on board grids. The programmer defines grid size and color (number of rows, number of columns, color of grid). The challenge is to create a picture that contains many squares with layers coming from a single point on the grid.

Topics

  • 2D arrays
  • Nested Loops
  • Conditionals

Intended learning outcomes

By the end of this project, students will be able to…

  • Apply: Represent branching logical processes by using conditional statements. (AP CS A Unit 3, CON-2.C )
  • Apply: Use understanding of row and column while creating algorithms for nested iteration. (AP CS A Unit 4, CON-2.C, CON-2.E, CON-2.G )
  • Apply: Use understanding of “out of bounds” errors when debugging code. (AP CS A Unit 8, CON-2.N )
  • Apply: Use an API as reference in order to figure out how to use Java library classes.
  • Apply: Use an API and class headers as reference when determining inheritance relationships. (AP CS A Unit 9, MOD-3.B )
  • Create: Apply above described cognitive skill to avoid out-of-bounds errors and test for requirements when generating intricate repeating patterns within a 2D grid.
